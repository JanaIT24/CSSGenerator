MULTIPLE COLUMN

The multiple-column layout specification provides you with a method for laying content out in columns,as like in the news paper.Using the `column-count` property in css allows you to break up a long paragraph into a specified number of columns.For instance,setting `column-count:3;`divides the content into three columns.

Syntax:column-count:value;column-gap:value;column-rule:width style color
